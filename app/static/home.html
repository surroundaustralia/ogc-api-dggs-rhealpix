DGGS OGC API - rHEALPix implementation

This API partially implements the OGC API DGGS specification.

Please see the automatically generated OpenAPI documentation at docs for details on how to use this API. Alternate documentation is available at redoc.

It utilises the rhealpix-sf and rhealpix-geo for spatial calculations.

The rhealpix-geo library borrows from (but does not depend on) rHEALPixDGGS for some low level Cell relationships.
